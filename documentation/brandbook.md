**Visual Design Direction & Aesthetic Brief**  
Project codename: **Mono Pulse** (or **Clean Pulse** / **Orange Edge**)

**Core Philosophy**  
Clean. Confident. Premium-minimal.  
We are building expensive-feeling minimalism that doesn't shout — it quietly owns the screen. The interface should breathe, feel spacious, and still look sharp and modern — imagine Teenage Engineering, Nothing, Revolut and Notion had a child raised in a Scandinavian black-box studio.

**Primary Inspirations**  
- **Teenage Engineering** — raw prototype honesty, stripped-back industrial minimalism, playful-yet-serious Scandinavian restraint, functional beauty  
- **Nothing** — transparent logic, monochromatic base + bold single-accent courage, retro-futuristic dot-matrix hints, glyph-like clarity  
- **Notion** — ruthless content-first clarity, modular structure, extreme airiness, hierarchy through space not decoration  
- **Revolut** — banking-grade confidence, razor-sharp precision, high information density without visual noise, “everything under control” vibe

**Mood Board Keywords**  
premium minimalism · tech austerity · functional elegance · dark techno · quiet dominance · sharp restraint · nothing-to-prove confidence · late-night prototype aesthetic

**Color System** — strict monochrome + one hero accent  
- True Black base (#000000 or #0A0A0A / #111111)  
- Surface / cards: #121212 → #1A1A1A  
- Primary text: #F5F5F5 / #EDEDED  
- Secondary / tertiary text: #8A8A8F → #A0A0A5  
- **Brand accent**: vivid orange #FF5E00 (or slightly desaturated #FF6B1A / #FF7426)  
  → used surgically: primary CTAs, active states, progress bars, key icons, badges, selected items  
- Error: muted red #FF2D55 (very rare)  
- Success: orange or pure white  
→ No gradients (except subtle vignette in rare hero moments). No secondary colors.

**Typography**  
- Primary face: **Inter** / **Neue Haas Grotesk** / **GT Walsheim** (cleanest neo-grotesque possible)  
- Fallback: **Manrope** or system **SF Pro**  
- Weights: Regular/Medium for body, SemiBold/Bold for headings & emphasis  
- Scale: strict 4-point grid (4, 8, 12, 16, 20, 24, 32, 40, 48 px)  
- Line height: 1.32–1.45× for breathing room  
- Alignment: left by default, center only for hero titles

**UI Language & Atmosphere**  
- Extreme **negative space** — generous padding (24–48 px on most screens)  
- Elements float with subtle depth: 1 px borders (#222–#333) or very soft shadows  
- Corner radius: 12–16 px cards/buttons, 8–10 px small elements, 20–24 px large panels  
- Icons: 1.5–2 px line weight, outline preferred, filled only on active/hero states  
- Micro-interactions: short (120–180 ms), buttery-smooth (cubic-bezier 0.4 0 0.2 1), purposeful, never decorative  
- Contrast: WCAG AA+ minimum, even on deepest blacks  
- No textures, no noise, no skeuomorphism, no neons/glows  
- Animations feel “alive” but restrained — nothing bounces or overshoots for fun

**What we explicitly avoid**  
- rainbow / multi-color accents  
- large illustrations or 3D assets  
- glassmorphism / heavy blur  
- cyberpunk / neon / retro overload  
- cute / playful microcopy elements  
- excessive motion / parallax  
- warm off-whites or light mode (dark-only unless later requested)

**One-sentence gut-check**  
When someone opens the app, it should feel like:  
“Expensive Swedish hi-fi brand meets banking fintech that doesn’t need to prove anything anymore.”
