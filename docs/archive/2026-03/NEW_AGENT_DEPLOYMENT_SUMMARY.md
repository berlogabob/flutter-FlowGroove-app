## 🎯 NEW CONTROLS

### Master Agent Commands

#### mr-supervisor:
```bash
/supervisor status              # Get overall agent status
/supervisor conflicts           # List active conflicts
/supervisor escalate [issue]    # Escalate to user
/supervisor reassign [task] [agent]  # Reassign task
/supervisor suspend [agent]     # Suspend agent for violations
```

#### mr-compliance:
```bash
/compliance audit [agent/file/task]  # Audit compliance
/compliance violations               # List recent violations
/compliance warn [agent]             # Issue warning
/compliance block [merge/release]    # Block for compliance
/compliance report                   # Generate compliance report
```

#### mr-quality-control:
```bash
/qc check [PR/release]      # Run quality gate check
/qc block [merge/release]   # Block for quality issues
/qc approve [merge/release] # Approve after checks pass
/qc metrics                 # Show quality metrics
/qc retrospective [version] # Run quality retrospective
```

## 🎯 HOW TO USE NEW AGENTS

### Example 1: Start New Feature
```
User: @mr-planner Design song structure editor

mr-planner: Decomposes task
mr-sync: Assigns to mr-architect, mr-ux-agent
mr-architect: Designs architecture
mr-ux-agent: Implements UI
mr-senior-developer: Reviews code
mr-theme-guardian: Checks theme compliance
mr-tester: Writes tests
mr-compliance: Verifies all rules followed
mr-quality-control: Final quality gate
mr-supervisor: Approves for merge
```

### Example 2: Rule Violation
```
Agent Issue: mr-cleaner refactored without user request

mr-compliance: Detects violation → Issues warning
mr-supervisor: Reviews → Reassigns task → Monitors
If repeat: mr-supervisor suspends agent → Notifies user
```

### Example 3: Quality Escape
```
Bug Found: Production bug in release

mr-quality-control: Investigates → Identifies failed gate
mr-supervisor: Calls retrospective → Process fix
mr-compliance: Updates rules → Prevents recurrence
```
